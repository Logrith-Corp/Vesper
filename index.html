What makes Vesper different from Chrome or Arc?
Vesper replaces the interaction model entirely. Traditional browsers wait for your input. Vesper's engine runs ahead of it.
Is Vesper really private — no telemetry at all?
Architecturally yes. Privacy core is open source. No analytics, no heartbeat, no crash reports unless you opt in.
What hardware does local AI require?
Apple Silicon M1+ or any modern x86 with 8GB RAM. GPU acceleration used automatically when available.
Is Vesper free?
Core browser is free forever. Vesper Pro unlocks advanced AI, E2E sync, and priority model updates.
When is Windows / Linux releasing?
macOS beta is live now. Windows in Q2 2026. Linux alongside Windows.
How does Neural Prefetch work without tracking me?
Neural Prefetch runs entirely on-device using local session context. No data ever leaves your machine.
